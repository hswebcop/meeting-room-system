⚙️ 系統設定指南

1
建立 Google Sheet
前往 Google Sheets 新增試算表，複製網址中的 ID（/d/YOUR_ID/edit）
2
建立 Google Apps Script
在試算表中點選「擴充功能」→「Apps Script」，貼上 Code.gs 的內容，將 SPREADSHEET_ID 換成您的 Sheet ID。
3
執行 setup() 初始化
在 Apps Script 中選取 setup 函式並執行，授予試算表與 Gmail 權限。
4
部署為 Web App
點選「部署」→「新增部署」，類型選「Web 應用程式」，執行身分選「我」，存取層級選「任何人」，複製部署網址。
5
填入 API 網址
將部署網址貼到下方，儲存後即可使用系統。
